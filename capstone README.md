# 📊 Báo Cáo Chiến Lược & Kiểm Toán Hiệu Quả Đầu Tư L&D — TechViet Corporation

Dự án Dashboard báo cáo phân tích định lượng tương tác cao (Bilingual Web Interactive Application), áp dụng mô hình kiểm soát **Phillips ROI Level 5** cô lập biến số ngoại cảnh dựa trên tập dữ liệu kiểm toán 200 bản ghi nhân sự tại TechViet Corporation. Giao diện được phát triển theo ngôn ngữ cấu trúc kể chuyện **Đường cong kịch tính Freytag (Freytag's Arc)** chuẩn McKinsey/MBB.

## 🚀 Tính Năng Nổi Bật của Dashboard
- **Executive Summary View**: Góc nhìn tóm lược dữ liệu mật độ cao dành riêng cho Hội đồng Quản trị và CFO với cấu trúc chuẩn ScR (*Situation - Complication - Resolution*).
- **Strategic Deep-Dive View**: Mạch phân tích 5 chương chi tiết dẫn dắt từ bối cảnh, mâu thuẫn ngân sách, điểm bùng nổ tài chính ròng 123.7 tỷ VNĐ cho đến giải pháp thực thi.
- **Bilingual Dynamic Engine**: Cơ chế chuyển đổi ngôn ngữ song ngữ song song mượt mà (**English | Vietnamese**) cho 100% nội dung văn bản và nhãn dữ liệu biểu đồ.
- **Premium Glassmorphism Design**: Thiết kế giao diện hiệu ứng kính mờ tối tân trên nền tối chuyển màu sâu kết hợp các điểm nhấn dải sáng Neon hiện đại.

## 🛠️ Công Nghệ Sử Dụng
- **HTML5 & SVG Elements**: Cho cấu trúc giao diện và sơ đồ đường cong kịch tính.
- **Tailwind CSS (CDN)**: Quản lý kiến trúc CSS tiện ích và phân bổ lưới Responsive đa thiết bị.
- **Chart.js**: Động cơ hiển thị các đồ thị trực quan hóa dữ liệu (Bar, Radar, Horizontal Bar chart).

## 📂 Hướng Dẫn Deploy Lên GitHub Pages (3 Bước Siêu Tốc)
1. **Tạo Repository mới** trên GitHub của bạn (Ví dụ đặt tên là: `ld-roi-audit`).
2. **Upload toàn bộ các file** (`index.html` và `README.md`) lên nhánh `main`/`master` của kho lưu trữ vừa tạo.
3. Vào mục **Settings (Cài đặt)** > **Pages** của Repository:
   - Tại phần *Build and deployment*, chọn mục *Source* là **Deploy from a branch**.
   - Tại phần *Branch*, chọn **main** (hoặc `master`) và thư mục gốc là `/ (root)`, sau đó nhấn **Save**.
   - Đợi 1-2 phút, GitHub sẽ cung cấp cho bạn một đường link trực tuyến công khai để truy cập trực tiếp từ trình duyệt!
